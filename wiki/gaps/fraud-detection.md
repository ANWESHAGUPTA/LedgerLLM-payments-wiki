# Fraud Detection

## Status
This concept is mentioned in sources but not explained in enough depth for a full article. Needs dedicated sources.

## What we know so far
Stripe has fraud detection systems that can decline payments suspected to be fraudulent. When this happens, merchants should present it as a generic decline rather than revealing the fraud detection to avoid alerting potential fraudsters.
